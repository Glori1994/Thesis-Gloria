
This README explains what the content of this folder is:

(1) Raw data:
       
    1995-1996 benthic and coral cover information data source:

       Villamizar, E. (2000). Estructura de una comunidad arrecifal en Falcón,        Venezuela, antes y después de una mortalidad masiva. Revista de                Biologia Tropical, 47, 19–30

   2017 benthic and coral cover information data source:

       Photoquadrats and Photoquad Layers.

(2) Matrix "Cobertura 1996 2017"
   
   The matrix were data from Villamizar et al., 2000 and Miyazawa et al., 2017    is grouped.


(3) Scripts

    Scripts for obtainig information about Playa Mero cover 2017 but     substracting the information from Miyazawa et al., unpublished matrix.

    Scripts for generating graphics."Graficas cobertura historia.R",                                     "Cobertura historia.Rproj"

(4) Graphics

    The graphic products from this data. "Historical cover english. png"

(5) README_Historical cover

    A readme containing the information about data transformation processes.
